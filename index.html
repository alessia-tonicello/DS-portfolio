Netflix Ratings Analysis (Jan-Jun 2023)

Analysis of Netflix's top 10 highest and least-rated titles Jan to Jun 2023. It examines patterns in ratings, genres, and release years for both high and low-rated titles. Data cleaning, analysis, and visualization are performed using Python to extract meaningful insights.

California Housing Prices Analysis

Analysis of California housing prices using Python and the StatLib dataset. This project includes data preprocessing, handling missing values, and exploratory data analysis (EDA) through visualizations to identify key trends.

UNICEF Dataset Analysis

This project analyzes a UNICEF dataset with global indicators like GDP per capita, life expectancy, and dependency ratios. Using Python and libraries like Pandas, Matplotlib, and Seaborn, it explores data trends and insights into global socioeconomic patterns.
